Consejos o tips para mejorar nuestra memoria

¿Que es la memoria?es una función del cerebro que permite al organismo codificar, almacenar y recuperar la información del pasado. La memoria permite retener experiencias pasadas y, según el alcance temporal, se clasifica convencionalmente en: memoria a corto plazo

Ejercicios y recomendaciones para la memoria

  1. Utiliza mapas mentales para el aprendizaje.
  2. Mantener un cerebro activo.
  3. Suma información a tu mente.
  4. Estimula los sentidos.
  5. Asocia los hechos con imágenes.
  6. Dile hola al ejercicio físico.
  7. Practica la improvisación.
  8. Realiza cálculos mentales
  9. Haz ejercicios para la atención focalizada
  10. Lleva acabo ejercicios de atención sostenida

Alimentos que mejoran la mente

  • Betabel para regular el ritmo cardiaco
  • Pescado para mantener sanas las células cerebrales.
  • Frutas silvestres para prevenir fallos en las neuronas.
  • Proteína sin grasa para mejorar las habilidades cognitivas.
  • Nueces para mejorar la memoria.
  • Café para mejorar el enfoque.
